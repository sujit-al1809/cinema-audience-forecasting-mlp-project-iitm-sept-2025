# Cinema Audience Forecasting Challenge

This repository contains my solution for the **Cinema Audience Forecasting Challenge** hosted by IITM BS.

## Problem Statement
Predict daily theatre audience counts across multiple locations using machine learning models.

---

## Approach

- Data preprocessing and cleaning
- Feature engineering
- Baseline ML model (Linear Regression / Random Forest)
- Prediction generation for test dataset
- Submission file creation

---

## Dataset

- `train.csv` – training data
- `test.csv` – test data
- `sample_submission.csv` – submission format

---

## Project Structure

- `notebooks/` → Kaggle notebook
- `src/` → modular ML code
- `outputs/` → predictions
- `data/` → datasets

---

```bash
pip install -r requirements.txt
